# CiteFlow

## Назва проєкту

**CiteFlow — вебсервіс для створення та керування бібліографічними посиланнями**

## Студент

**Курка Женя**

## Опис проєкту

CiteFlow — це навчальний проєкт вебсервісу для автоматичного створення, редагування, копіювання, збереження та експорту бібліографічних посилань. Проєкт розробляється на основі аналізу сервісу-прототипу YesCite.

Основна мета системи — спростити процес оформлення списку використаних джерел для студентів, викладачів, науковців і авторів навчальних або наукових робіт.

## Сервіс-прототип

Для аналізу було використано вебсервіс:  
https://yescite.com/

## Основні можливості майбутньої системи

- створення цитування за DOI;
- створення цитування за URL;
- створення цитування за ISBN;
- автоматичне визначення типу введених даних;
- вибір стилю цитування;
- попередній перегляд сформованого посилання;
- ручне створення цитування;
- редагування метаданих джерела;
- копіювання готового цитування;
- експорт цитувань;
- реєстрація та авторизація користувачів;
- збереження цитувань до обраного;
- створення проєктів зі списками джерел.

## Актори системи

1. **Незареєстрований користувач** — може створювати, переглядати, копіювати та експортувати цитування.
2. **Зареєстрований користувач** — може зберігати цитування, створювати проєкти та додавати джерела до обраного.
3. **Адміністратор** — керує системою, користувачами та підтримуваними стилями цитування.
